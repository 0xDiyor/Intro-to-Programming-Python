## About This Repo

Coursework repository for **CSC 1019: Introduction to Programming – Python** at CCD (Spring 2026), using zyBooks as the learning platform.

- `labs/` — zyLabs exercises (shorter, topic-focused)
- `projects/` — larger course projects

## Running Python Files

```bash
python3 <file.py>
```

No build step, no dependencies, no virtual environment — this is a plain Python 3 course repo. Scripts are self-contained and run directly.

## Course Topics

Python fundamentals in this order: variables/expressions, types, branching, loops, functions, strings, lists/dictionaries, classes, exceptions, modules, files, inheritance, recursion, data science basics, searching/sorting algorithms.

## Conventions

- Files go in `labs/` or `projects/` depending on their source
- Each lab or project should live in its own subdirectory or be named clearly (e.g., `labs/ch3_types.py`, `projects/project1/`)
